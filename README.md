# Villa Sentoza — static landing page

Koncepcyjna strona wizytówka dla Villa Sentoza w Sopocie, przygotowana jako statyczny projekt HTML/CSS/JS.

## Stack

- `index.html`
- `styles.css`
- `script.js`
- GitHub Pages workflow

## Funkcje

- responsywny landing page,
- przełącznik języka PL/EN,
- mobilne menu,
- filtrowanie pokoi,
- demo formularza kontaktowego z walidacją,
- animacje wejścia sekcji,
- dane kontaktowe i linki do rezerwacji.

## Uwaga

To strona koncepcyjna oparta na publicznych informacjach o obiekcie. Nie używa płatnych bibliotek ani zewnętrznych fontów.
